# v1.3.1

## Summary
This patch tightens the `Stock Details` composition controls, standardizes holding category labels around Yahoo Finance sector metadata, and adds a dedicated multi-category preview workflow without touching the canonical holdings file.

## Stock Details Composition Refinement
- Removed the extra composition heading block so the section now reads as a compact percentage bar with filter chips.
- Reworked the category chips into fixed-width bilingual controls with a tighter vertical rhythm, left alignment, and wrap support on narrow layouts.
- Reduced repeated color emphasis by keeping the strongest color signal in the composition bar while softening chip backgrounds, borders, and percentage accents.

## Category Mapping Basis
- Standardized display labels around Yahoo Finance `sector` and `industry` metadata instead of theme-style names such as `科技成長`.
- Expanded the local category map to cover standard sector-style buckets including `Consumer Defensive`, `Basic Materials`, and `Utilities`.
- Documented the intended classification basis through tests, including `GOOG` resolving to `Communication Services` because Yahoo Finance profiles it under `Communication Services / Internet Content & Information`.

## Preview And Verification
- Added a `/tmp` preview flow using `PORTFOLIO_HOLDINGS_PATH` so broader multi-category layouts can be reviewed without modifying `data/holdings.json`.
- Verified with:
  - `PYTHONPYCACHEPREFIX=/tmp/pycache python3 -m py_compile portfolio.py`
  - `PYTHONPATH=. .venv/bin/pytest`
  - `python3 portfolio.py --output docs/index.html`
  - Local review of `http://127.0.0.1:5002`, `/health`, and a `/tmp` static preview build
