# v1.3.5

Date: 2026-05-03

This patch adds a lightweight `Project Roadmap` overlay to the site header so the current version milestones and next implementation phases are visible from the live portfolio itself.

## Project Roadmap Overlay

- Added a small roadmap trigger beside the main portfolio title in the header.
- Added a centered overlay panel that keeps the current page visible underneath instead of taking over the full screen.
- Split the roadmap into two vertical sections:
  - completed major version milestones with checked markers
  - upcoming roadmap items in the planned implementation order
- Kept the roadmap content in configuration and snapshot context so the template stays presentation-focused.

## Interaction

- Added open / close behavior through the existing frontend bundle.
- Closing works by:
  - clicking the close button
  - clicking the backdrop around the panel
  - pressing `Escape`
- Locked page scroll while the roadmap is open so the overlay feels intentional on both desktop and mobile.

## Verification

- `PYTHONPYCACHEPREFIX=/tmp/pycache python3 -m py_compile portfolio.py`
- `python3 portfolio.py --output docs/index.html`
