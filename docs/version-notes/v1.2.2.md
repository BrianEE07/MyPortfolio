# v1.2.2

## Summary
This patch applies a small `Holdings Overview` polish pass so the hero P&L cards read more consistently and the secondary metrics match the intended order.

## Overview Label And Order Updates
- Updated `Unrealized` and `Realized` overview amounts to use signed currency formatting so positive values explicitly render with a leading `+` and losses render with `-`.
- Renamed the overview English labels from `P/L` to `P&L`, including the compact responsive labels used on smaller screens.
- Swapped the `Alpha` and `Beta` overview positions so the secondary metrics now read:
  - `Portfolio YTD`
  - `Sharpe Ratio`
  - `Alpha`
  - `Beta`
