# v1.1.5

## Summary
This version focused on responsive header cleanup and touch-friendly interactions in `Stock Details`.

## Header Responsive Fixes
- Reordered the header actions so `Last updated` appeared before the theme toggle.
- Kept `Last updated` on the left in side-by-side layouts and above the toggle in stacked layouts.
- Added `text-size-adjust` rules to reduce real-mobile font inflation.
- Added a small-screen landscape header rule so phone landscape layouts used a more compact subtitle treatment instead of the desktop fallback.

## Tooltip Interaction
- Reworked the help `?` tooltip behavior around a single active tooltip state.
- On touch screens, tapping a help chip toggled its tooltip and tapping outside closed it.
- Added `Escape`, scroll, resize, and tab-switch cleanup for lingering floating tooltips.
- Kept desktop hover and keyboard focus behavior intact.

## Stock Details Sorting
- Preserved the three-step sort cycle.
- Tightened the reset-to-default path so returning to `持倉市值 / Position Value` restored the active header styling and descending state cleanly.
- Limited inactive hover tint to hover-capable pointers so touch devices no longer left stale active-looking headers behind.

## Verification
- Rebuilt the static site successfully with `python3 portfolio.py --output docs/index.html`.
- Verified the Flask app still served `GET /` and `GET /health` successfully.
