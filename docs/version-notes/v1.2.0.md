# v1.2.0

## Summary
This version refreshes `Holdings Overview` so the page emphasizes the four hero portfolio numbers first, softens the advanced metrics into a lighter stats rail, and generates `Realized P/L` directly from Firstrade imports.

## Holdings Overview Refresh
- Replaced the previous flat 8-card overview with a two-tier layout:
  - primary cards for `Market Value`, `Total Cost`, `Unrealized P/L`, and `Realized P/L`
  - secondary cards for `Portfolio YTD`, `Sharpe Ratio`, `Beta`, and `Alpha`
- Removed the bottom supporting copy from all eight stats so the overview stays focused on the most important numbers.
- Moved P/L percentages inline to the right of the main amount on the two hero P/L cards.
- Restyled the four secondary metrics into a lighter transparent stats rail instead of full cards.
- Kept the mobile summary as a compact two-column layout with tighter vertical spacing so more cards remain visible on one screen.

## Metrics And Generated Portfolio Data
- Added generated canonical support for `data/portfolio_metrics.json` with:
  - `realized_pl`
  - `realized_return_pct`
- Firstrade imports now compute realized performance from the same average-cost trade reconstruction used for current holdings.
- Canonical CSV and canonical JSON imports reset generated realized metrics so stale FT realized data is not preserved.
- Promoted `Alpha` into the overview and formatted it as a percentage-style excess-return value instead of the previous raw decimal-style output.
- Added bilingual info-chip definitions for `Portfolio YTD`, `Sharpe Ratio`, `Beta`, and `Alpha`.

## Top Holdings And Mobile Polish
- Renamed the holdings chart section to `Top 10 Holdings`.
- Replaced the old subtitle sentence with a single `Share` card that lists `Top 3`, `Top 5`, and `Top 10` market-value concentration.
- Refined the floating mobile tab bar toward a softer, more transparent liquid-glass look.
- Reduced the bottom disclaimer text to `80%` scale, switched the order to Chinese first, and split the two lines in phone portrait.
- Added a more defensive chart relayout path so rotation into phone portrait no longer needs a manual refresh to recover the `Top 10 Holdings` layout.
