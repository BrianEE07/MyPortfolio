# v1.3.0

## Summary
This version completes the first `Market Pulse` refresh, adds portfolio composition filtering to `Stock Details`, and ships a standard favicon direction for the site.

## Portfolio And Icon Updates
- Added a standard pie-chart and money-bag favicon direction for browser and Apple touch icons.
- Updated the `Top 10 Holdings` chart so holdings after the tenth row are grouped into `Others`.
- Added tooltip formatting for holdings chart values with `$` currency and percentage share.

## Stock Details Composition
- Added a portfolio composition bar above the holdings table.
- Normalized Yahoo Finance sector and industry metadata into stable local holding categories.
- Added category filtering while preserving the active table sort and recalculating top-market-value rank highlights within visible rows.

## Market Pulse Refresh
- Reworked the page into three blocks: `Market Sentiment`, `Market Trend`, and `Dip Signals`.
- Added a compact fear-and-greed gauge, market trend chart, valuation strip, and source icon links.
- Moved explanatory copy for dip-signal metrics into concise info cards.
- Added a FINRA margin-debt fallback chain of live HTML parsing, official XLSX download parsing, and a final built-in payload so temporary request blocking does not render the value as `N/A`.
