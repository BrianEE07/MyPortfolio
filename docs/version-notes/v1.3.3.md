# v1.3.3

## Summary
This patch refreshes the browser and Apple touch icon assets to remove the dark rounded-corner canvas from the previous favicon set.

## Icon Updates
- Rebuilt the favicon direction on a full-bleed light background so browser tabs and iOS home-screen icons no longer expose black corners.
- Added a standard multi-size `favicon.ico`, a 192px web app icon, and a web manifest.
- Kept the browser favicon path on raster assets so the generated liquid-glass artwork remains consistent across browsers.
- Kept `apple-touch-icon.png` at the expected 180x180 size with no alpha channel so iOS can apply its native rounded mask.

## Verification
- Verified generated PNG dimensions and alpha state with `sips`.
- Checked favicon edge luminance to confirm the outer pixels are light instead of dark.
