# v1.1.3

## Summary
This version refined the responsive header and introduced the floating mobile-tab navigation.

## Header And Theme Toggle
- Reworked the hero header so the title block stayed primary while the theme toggle and update metadata remained pinned in the top-right.
- Replaced the old update card with an inline English-only `Last updated` block.
- Improved spacing and alignment for both side-by-side and stacked header layouts.
- Replaced emoji-based theme icons with monochrome inline SVG.
- Tightened the toggle into a smaller pill while preserving the light/dark state indicator.

## Floating Mobile Tabs
- Restyled the desktop tabs into equal-width bookmark-style tabs with lower visual weight.
- Added a bottom floating tab control for tablet and mobile breakpoints.
- Hid the large desktop tab cards on smaller screens so overview content appeared earlier.
- Added bottom page padding so the floating control did not cover content during scroll.
- Removed size-changing active states to avoid jitter while switching tabs.

## Verification
- Rebuilt the static site successfully with `python portfolio.py --output docs/index.html`.
- Verified the Flask app still served `GET /` and `GET /health` successfully.
- Confirmed the rendered HTML included both desktop and floating tab controls and the English-only `Last updated` label.
