# v1.3.2

## Summary
This patch finishes the `Stock Details` category-chip interaction polish so narrow layouts reset cleanly and keep the selected filter visible.

## Category Filter Interaction
- Kept the compact chip treatment from `v1.3.1` while preserving the active border in horizontally scrollable layouts.
- Auto-scrolls the selected category chip fully into view when the user picks a filter that sits outside the current viewport.
- Resets the category filter back to `全部` when leaving `Stock Details`, and also resets the chip rail scroll position to the default start state before the user returns.
