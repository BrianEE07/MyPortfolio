# v1.3.4

## Summary
This patch refines the `Market Sentiment` block in `Market Pulse` after the first browser review pass.

## Market Sentiment
- Changed Fear & Greed score display to truncate official CNN values so the app matches CNN's visible integer labels.
- Changed the Fear & Greed day-over-day badge to compare the truncated display scores, with matching display values rendered as a flat state.
- Updated the previous close, one-week, one-month, and one-year comparison values to prefer CNN's official summary fields.
- Kept the Fear & Greed chart on CNN's raw historical series while labeling the card value as the official previous close.
- Rebuilt the Fear & Greed gauge as an SVG half-dial so the colored bands and pointer stay aligned across viewport sizes.
- Dimmed non-current gauge bands while keeping the active score band at full color.
- Replaced the broad category text row with compact numeric score ranges and full Chinese labels aligned to the gauge width.
- Rebalanced the sentiment block into four responsive modules: current score, gauge, trend chart, and compact historical comparison, with the desktop layout keeping the current label beside the delta badge and capping chart dominance on wide screens.
- Tuned the sentiment chart region so the gauge and trend chart keep an approximate 40/60 width relationship across tablet and desktop layouts.
- Switched the compact historical comparison to a single-column list on narrow mobile screens so full Chinese labels remain readable without abbreviating sentiment text.
- Reduced the Fear & Greed history chart x-axis tick density to keep date labels readable.
- Reworked previous-close, one-week, one-month, and one-year values into a compact comparison strip.
