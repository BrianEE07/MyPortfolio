# v1.3.4

## Summary
This patch refines the `Market Pulse` page after the first browser review pass, with focused work on sentiment, trend, and dip-buying indicators.

## Market Sentiment
- Changed Fear & Greed score display to truncate official CNN values so the app matches CNN's visible integer labels.
- Changed the Fear & Greed day-over-day badge to compare the truncated display scores, with matching display values rendered as a flat state.
- Updated the previous close, one-week, one-month, and one-year comparison values to prefer CNN's official summary fields.
- Kept the Fear & Greed chart on CNN's raw historical series while labeling the card value as the official previous close.
- Rebuilt the Fear & Greed gauge as an SVG half-dial so the colored bands and pointer stay aligned across viewport sizes.
- Dimmed non-current gauge bands while keeping the active score band at full color.
- Added a requestAnimationFrame needle entrance animation that draws the pointer from the left edge to the current score by updating the SVG line endpoint, with the active gauge band and range label following the animated score.
- Replaced the broad category text row with compact numeric score ranges and full Chinese labels aligned to the gauge width.
- Rebalanced the sentiment block into four responsive modules: current score, gauge, trend chart, and compact historical comparison, with the desktop layout keeping the current label beside the delta badge and capping chart dominance on wide screens.
- Tuned the sentiment chart region so the gauge and trend chart keep an approximate 40/60 width relationship across tablet and desktop layouts.
- Switched the compact historical comparison to a single-column list on narrow mobile screens so full Chinese labels remain readable without abbreviating sentiment text.
- Reduced the Fear & Greed history chart x-axis tick density to keep date labels readable.
- Reworked previous-close, one-week, one-month, and one-year values into a compact comparison strip.
- Retitled the section to `CNN Fear & Greed Index · 市場情緒`.
- Expanded the Fear & Greed trend chart to the one-year CNN series.
- Added consistent chart tooltip spacing and a vertical hover guide line.
- Changed Fear & Greed chart labels to include year, month, and day in the hover title.

## Market Trend
- Reworked the Market Trend block so the S&P 500 current price is the first visual signal.
- Added a trend status summary derived from 20-day, 60-day, and 250-day moving-average breaks.
- Added distance-from-moving-average values for 20MA, 60MA, 250MA, and 1250MA signal cards.
- Added 20MA, 60MA, and 250MA reference lines to the S&P 500 trend chart, with the price line color tied to the current trend status.
- Retitled the section to `S&P 500 Technical Trend · 大盤趨勢`.
- Removed the block treatment from the current-price hero so the primary price reads like the sentiment score.
- Compactly rendered trailing P/E values in two-line cards.
- Changed the moving-average signal cards to a vertical stack on wide screens so the technical chart gets more width.
- Shortened the moving-average signal cards and vertically centered the stack beside the technical chart on wide screens.

## Dip-Buying Indicators
- Retitled the section to `Dip-Buying Strategy Indicators · 抄底訊號`.
- Moved indicator definitions into the question-mark tooltips.
- Removed status tags from the card headers because the current-zone field now carries that meaning.
- Added current-zone and dip-buying interpretation rows to each indicator card.
- Added threshold tooltips to current-zone rows so the zone labels explain their own criteria.
- Moved FINRA margin debt latest month and month-over-month change beside the headline number.
- Changed current-zone threshold tooltips to a bullet-style format.
- Equalized the current-zone and dip-buying interpretation cells so wrapping text does not create uneven card interiors.
- Restored multiline tooltip wrapping so longer indicator explanations can break naturally.
