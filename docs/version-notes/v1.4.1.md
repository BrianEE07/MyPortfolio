# v1.4.1

Date: 2026-05-08

This release polishes the v1.4.0 overview and market-pulse UI after mobile review.

## Overview Polish

- Replaced the title-side roadmap trigger with a route-style icon.
- Added the `View Roadmap` tooltip and fixed the tooltip dismissal when opening the roadmap dialog.
- Tightened mobile landscape summary cards so P&L percentages stay inside their cards.
- Hid the horizontal scrollbar for the Stock Details table while preserving touch scrolling.

## Market Pulse Polish

- Preserved the four-column Fear & Greed history layout on small landscape screens while fixing the broken border treatment.
- Added `較上月` beside the S&P 500 trailing P/E delta.
- Kept the trailing P/E delta note aligned with the numeric delta without pixel-based vertical offsets.
- Added a dollar prefix and the `百萬美元` unit to FINRA margin debt while keeping the unit visually secondary.
